# 📄 PDF 페이지 병합기

여러 PDF 파일에서 원하는 페이지만 골라 하나의 PDF로 합쳐주는 브라우저 기반 도구입니다.

> Made by **공하은**

---

## 시작하기

1. `pdf-page-merger.html` 파일을 다운로드합니다.
2. 브라우저(Chrome, Edge, Firefox 등)에서 파일을 엽니다.
3. 끝입니다. 설치할 것도, 로그인할 것도 없습니다.

---

## 주요 기능

### 다중 PDF 업로드
파일을 드래그 앤 드롭하거나 클릭하여 여러 PDF를 한번에 추가할 수 있습니다.

### 페이지 썸네일 미리보기
업로드한 PDF의 모든 페이지가 작은 이미지로 표시됩니다. 어떤 페이지에 어떤 내용이 있는지 눈으로 확인하면서 작업할 수 있습니다.

### 유연한 페이지 선택

| 방법 | 설명 |
|------|------|
| `1-5` | 1페이지부터 5페이지까지 |
| `3, 7, 12` | 개별 페이지 지정 |
| `1-3, 7, 10-15` | 범위와 개별 혼합 |
| `all` | 전체 페이지 |
| 썸네일 클릭 | 클릭한 페이지 선택/해제 |
| Shift + 클릭 | 시작~끝 범위 일괄 선택 |

텍스트 입력과 썸네일 클릭은 실시간으로 연동됩니다.

### 선택 해제
- 각 파일별 **선택 해제** 버튼: 해당 파일의 페이지 선택만 초기화 (파일은 유지)
- 하단 **전체 선택 해제** 버튼: 모든 파일의 선택을 일괄 초기화

### 페이지 번호 삽입
병합된 PDF에 자동으로 페이지 번호를 넣을 수 있습니다.
- **위치**: 하단 중앙 / 하단 우측 / 상단 중앙 / 상단 우측
- **시작 번호**: 1이 아닌 다른 숫자부터 시작 가능
- **크기**: 작게 / 보통 / 크게

### 저장 파일명 지정
병합 전에 원하는 파일명을 입력하면 해당 이름으로 PDF가 저장됩니다.

### 다크 / 라이트 모드
우측 상단 토글로 테마를 전환할 수 있습니다.

## 보안 및 프라이버시

- 모든 처리는 **브라우저 내부**에서만 이루어집니다.
- 업로드한 파일은 **외부 서버로 전송되지 않습니다.**
- 네트워크 연결 없이 **오프라인**으로 사용 가능합니다.
- 보안이 민감한 문서도 안심하고 작업할 수 있습니다.

## 사용 예시

**기말 자료 정리**
> 교재A의 3~5장, 교재B의 요약 페이지, 과제C의 표지만 뽑아서
> 하나의 제출용 PDF로 엮기

**업무 보고서 편집**
> 여러 부서에서 받은 PDF 중 필요한 페이지만 골라
> 하나의 회의 자료로 병합하기

**스캔 문서 재구성**
> 스캔된 PDF에서 필요한 페이지만 추려
> 새로운 문서로 만들기

## 기술 스택

- **pdf-lib** — PDF 생성 및 페이지 병합
- **PDF.js** (Mozilla) — 페이지 썸네일 렌더링
- 모든 라이브러리가 HTML 파일 안에 내장되어 있어 단일 파일로 동작합니다.
